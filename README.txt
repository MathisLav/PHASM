Fonctionnalités :

	Voici PHASM, avec lui vous pourrez exécuter tous vos programmes de la même façon !
	De plus, il suffit d'appuyer sur [Alpha] [X,T,O,n] consécutivement pour afficher un menu qui vous permettra de choisir le Lbl auquel vous voulez accéder
	PHASM est de plus compatible avec CEsium ;)
	

Installation :

	Pour l'utiliser, il vous faut transferer le group PHASM et le programme PHASM.
	Ensuite, exécutez Asm(prgmPHASM) et le tour est joué !
	Pour désinstaller PHASM, suffit d'exécuter prgmPHASM.
	Ne supprimez ou n'archivez pas l'AppVar qui va être créé (ça vous désactivera le hook d'exécution des programmes assembleur et archivés).
	Cela ne sert à rien de dégrouper le group PHASM. 


Have fun!
